# Assets

## App icon

The repository includes:

- `assets/AppIcon-source.png`
- `assets/AppIcon.icns`
- iOS app icon PNGs under `ios/Resources/Assets.xcassets/`

These assets are part of Multiharness and are licensed under the repository license unless replaced by downstream distributors.

## Completion sound

`Sources/Multiharness/Resources/agent-ding.wav` is generated by `scripts/generate-ding.py`. See `Sources/Multiharness/Resources/CREDITS.txt` for details.

## Regenerating icons

Use:

```bash
bash scripts/generate-app-icon.sh
```

Generated intermediates under `assets/AppIcon.iconset/` are ignored.
