# Roadmap

This roadmap is intentionally lightweight. Detailed design notes live in `docs/superpowers/specs/` and implementation plans live in `docs/superpowers/plans/`.

## Near-term open-source readiness

- Keep CI green for Swift and sidecar tests.
- Publish screenshots and a short demo video/GIF.
- Add a signed and notarized macOS release workflow.
- Improve first-run onboarding and permissions explanations.
- Expand privacy/security documentation as remote access evolves.

## Good first issues

- Improve README screenshots and walkthrough copy.
- Add small UI polish with before/after screenshots.
- Expand troubleshooting entries from real user reports.
- Add tests around sidecar tools and path-guard edge cases.
- Improve provider setup help text.

## Larger projects

- TLS or a stronger transport story for remote access outside Tailscale.
- Token rotation and pairings management UX.
- Release automation for macOS and iOS builds.
- Richer diff/review UI and merge-conflict workflows.
- More provider integrations and model metadata.
